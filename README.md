# Media Keys

MOTE branch!

Gnome-shell extension that adds Media Player buttons to the panel.

#### !!! NEEDS xdotool

### Changelog

* v.1
  - First version

* v.2
  - Switched to mote branch!

### Notes

* v.1 Extension emulates buttons:
  - XF86AudioPlay
  - XF86AudioPrev
  - XF86AudioNext

* v.2 executes mote commands:
  - pause
  - prev
  - next
